# Emoji Keywords

Keyword to code mapping for [Twemoji.js](https://github.com/twitter/twemoji) (V2) which supports 1661 emojis.

Available as [YAML](/twemoji/v2/keywords.yaml), and [JSON](/twemoji/v2/keywords.json) format.
